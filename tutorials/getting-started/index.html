Getting started with the Core workspace

This tutorial is a placeholder for the first complete public walkthrough.

Planned outline

  1. Clone the repository and read the working agreement.
  2. Bootstrap local tooling.
  3. Run the standard verification commands.
  4. Open the module guide and map a behavior to its authority source.
  5. Make a small docs-only change and produce a completion report.

Verification commands

dotnet build NexusScholar.Core.slnx -c Release
dotnet test NexusScholar.Core.slnx -c Release --no-build
dotnet format NexusScholar.Core.slnx --verify-no-changes --no-restore
